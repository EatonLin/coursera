# Precision/Recall

## Precision(准确率)
$$
Precision = \frac{True\ positives}{\#\ predicted\ as\ positive} = \frac{True\ positives}{True\ positives + False\ positives}
$$
$$
Recall = \frac{True\ positives}{\#\ actual\ positive} = \frac{True\ positives}{True\ positives + False\ negatives}
$$
